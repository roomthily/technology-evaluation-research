RRL Criteria
1 Software or executable available only, no packaging. Inadequate or no documentation and no auto-build/install facility is available
3 Detailed installation instructions available. System includes auto-build feature, but is built for a particular operating system.
5 Software is easily configurable for different contexts. For example, locations of resources (files, directories, URLs) are configurable. All configurationspecific information is centralized.
7 OS-detect and auto-build for supported platforms available. Operating system detection configuration files are available. Packaging includes auto-build for supported OS platforms and suite of regression tests for platform-specific testing.
9 Installation user interface provided. A user interface guides the installer through all steps needed to build, configure, and install the software
